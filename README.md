A responsive multi-step form built using JavaScript that guides users through a step-by-step onboarding process with validation and state management.

## Features
- Multi-step navigation (Next / Previous)
- Form validation at each step
- Data persistence between steps
- Progress indicator
- Reset functionality

## Tech Stack
- HTML
- CSS
- JavaScript (Vanilla JS)

## Project Structure
- index.html
- style.css
- app.js

## How to Run
1. Clone the repository
2. Open `index.html` in your browser
